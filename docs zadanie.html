MOJE ZADANIE

Opis zdania:

Projekt służy do obliczania długości słow oraz zdań.

Dane:

Należy wpisać dowolne zdanie lub zbiór słów w podanym przez program miejscu.

Wynik

Pod napisanymi wcześniej słowami pokaże się występująca w nich ilość liter.

import java.util.Scanner;
public class abcd {
static Scanner sc = new Scanner(System.in);
int length(String s) {
return s.length(); }
public static void main (String [] args) {
abcd licz = new abcd();
System.out.println("Wpisz poniżej dowolny tekst:");
String s = sc.next();
System.out.println("Poniżej pokazuje się ilość znaków w tym wyrazie:");
System.out.println("długość:"+licz.length(s));
} }
